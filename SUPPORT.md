# Support

Use this guide to decide where to ask for help.

## Workshop Participants

For help during a live session, ask the workshop mentors or the AWS Cloud Club FAST Peshawar team member assigned to your session.

For repository issues after the session:

- use GitHub issues for public problems with docs, commands, examples, or setup steps
- include your operating system, session folder, command, and error output
- remove secrets, access keys, tokens, account IDs, and private student data before posting

## Maintainers and Volunteers

Use issues for work that should be tracked publicly:

- session documentation updates
- setup problems
- broken commands
- missing troubleshooting notes
- certificate generator bugs using sample data

Use the club's private maintainer channel for:

- leaked secrets
- real attendee data
- conduct concerns
- AWS account or billing problems involving a student
- certificate signing key rotation

## Asking a Good Question

Include:

- which session or tool you are using
- what you expected to happen
- what actually happened
- the exact command you ran
- the error message, with secrets removed
- what you already tried

## Response Expectations

This repository is maintained by student volunteers. Urgent security, privacy, or billing concerns should go through the official club leadership channel instead of waiting for a GitHub issue response.

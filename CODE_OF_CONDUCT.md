# Code of Conduct

AWS Cloud Club FAST Peshawar is a student community. This repository is part of that learning environment, so contributors are expected to act with respect, patience, and academic honesty.

## Expected Behavior

- be respectful to students, volunteers, mentors, and maintainers
- explain technical feedback clearly and constructively
- help beginners without mocking basic questions
- credit ideas, code, and references that come from others
- keep private club, student, and event information private
- follow university policies and event rules

## Unacceptable Behavior

The following behavior is not acceptable in issues, pull requests, discussions, commits, or workshop spaces:

- harassment, insults, threats, or discriminatory comments
- sharing private student information without permission
- posting AWS credentials, API keys, or other secrets
- encouraging unsafe cloud usage or avoidable billing risk
- plagiarism or misrepresenting someone else's work as your own
- spam, off-topic promotion, or repeated disruptive behavior

## Reporting Problems

For public documentation mistakes, open an issue.

For conduct concerns, do not open a public issue with personal details. Contact the club leadership or repository maintainers through the official AWS Cloud Club FAST Peshawar communication channel used for the event.

Reports should include:

- what happened
- where it happened
- when it happened
- links, screenshots, or usernames if available
- whether immediate action is needed

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, block contributors, or escalate serious concerns to club or university leadership. The goal is to keep the repository useful, safe, and welcoming for students.

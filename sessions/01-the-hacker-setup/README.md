# Session 01: The Hacker Setup

Environment setup for the AWS Silicon Valley Workflow workshop series.

## Focus

- Linux terminal basics
- WSL or Ubuntu setup
- Git and GitHub CLI
- Repository cloning and workflow hygiene

## Outcome

Students finish this session with a working local development environment and the command-line basics needed for the rest of the workshop.
